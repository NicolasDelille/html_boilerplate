Basic html boilerplate

Using

  • Gulp
  • Gulp icon font
  • Browserify
  • Babel [ES2015]
  • SASS w/ basic implementation
  • Home made mobile first CSS grid
  • Image optimization

Develop

$ npm start

Build

$ npm run build

Optim images

$ npm run images
